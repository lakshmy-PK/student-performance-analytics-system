# student-performance-analytics-system
Student Performance Analytics System is a Python project that uses Pandas to analyze student academic records stored in CSV files. The system provides insights into student performance through various analytical features.

FEATURES-
1.View Student Records
2.Find Class Topper
3.Calculate Class Average
4.Subject-wise Average Marks
5.Pass/Fail Analysis
6.Subject Toppers
7.Students Above Class Average
8.Export Analysis Report to CSV
9.Exit

TECHNOLOGIES USED-
1.Python
2.Pandas
3.CSV Files
